# Student Grade Manager

**Project #2 – Java Practice**

This is my second Java project. It’s a simple **Student Grade Manager** program that allows you to:  

- **Add Students** with a name and grade  
- **View Students** and their grades  
- **Update Student Grades**  
- **Delete Students**  
- **Calculate the Average Grade**  
- **Save and Load** data from a file (`tasks.txt`)  

## How to Use

1. Run the program.  
2. Choose an option from the menu (1–6).  
3. Follow the instructions to add, view, update, or delete students.  
4. Your data will be automatically saved to a file and loaded the next time you run the program.  

## Features

- Input validation for grades (0–100)  
- Automatic ID assignment for each student  
- Persistent storage using a text file
