My First Java Practice – Task Manager


A simple console-based Task Manager in Java. Add, view, complete, and delete tasks.
Tasks are saved to a file, so you can continue where you left off.
Just a practice project to learn Java basics.

1. Clone the repo
2. Open in your favorite Java IDE
3. Run TaskManagerApp.java

- Add new tasks
- View tasks with status
- Complete tasks
- Delete tasks
- Tasks saved to a file
