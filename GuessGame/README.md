# 🎯 Guess Game (Java)

This is my fourth Java project.

A simple number guessing game with different difficulty levels and a score system.

## 🚀 Features
- 3 difficulty levels:
  - Easy (1–50, 10 attempts)
  - Medium (1–100, 7 attempts)
  - Hard (1–200, 5 attempts)
- Hints:
  - Too high / Too low
  - "Very close" feedback
- Score system (based on remaining attempts)
- Replay option

## 🛠️ Technologies
- Java
- Console (CLI)
- Random & Scanner

## 🎮 How to Play
1. Choose a difficulty
2. Guess the number
3. Follow the hints
4. Try to win with the highest score!
